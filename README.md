# defi-mcp

An MCP server giving AI agents direct access to DeFi data on EVM chains.

**Status:** Early development. Not yet ready for public use.

## What it will do

Expose tools for any MCP-compatible LLM (Claude, etc.) to:
- Read wallet token balances across chains
- Analyze Aave v3 positions and health factors
- Track Uniswap v3 LP positions and impermanent loss
- Simulate swaps with realistic slippage
- Monitor protocol TVLs and oracle prices

## Supported chains

- Polygon (more soon)
